# StatsR

A structured R statistics repository containing:
- Regression modelling
- EDA (Exploratory Data Analysis)
- Blood pressure dataset analysis
- Multivariate datasets (Data16)

## Structure
data/raw/        -> datasets
scripts/         -> clean runnable R scripts
scripts/original -> original coursework scripts
docs/            -> PDFs and HTML notes

## Run examples
Rscript scripts/01_r_basics.R
Rscript scripts/02_regression_analysis.R
Rscript scripts/03_blood_pressure_analysis.R
